## Student Task Tracker
 a simple responsive Student Task Tracker built using HTML, Tailwind CSS, and vanilla JavaScript.

# Features
- Add new tasks with a due date
- Delete tasks when completed
- Form validation (no empty tasks allowed)
- Saves tasks using Local Storage
- Data stays after page refresh
- responsive design
# Tools Used
- HTML5
- Tailwind CSS
- JavaScript (DOM Manipulation)
- Local Storage
- Git & GitHub
# How to Use
1. Type your task name
2. Select a due date
3. Click "Add Task"
4. Click "Delete" to remove a task
